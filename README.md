
# Spark Note




## Run Locally

Clone the project

```bash
  git clone https://github.com/chaursiyasanjeet/Techvzero
```
# server setup

  ```bash
  cd backend
```
Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
# frontend

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file


##### Server Requirement

`MONGODB_URL`

`JWT_SECRET`

`JWT_EXPIRES`
